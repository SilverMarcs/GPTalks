# GPTalks

A SwiftUI multi-platform app to interact with OpenAI Chat Completions API along with some other providers.

The screenshots may also be often outdated. Feel free to grab the latest MacOS release from Github release page. iOS is self-compile only for now throigh XCode.

## Usage Instructions

Open Settings Menu on respective platform, choose your desired API provider and paste your API key and start chatting.
Click Image icon in start page toolbar for image generations. You may change the model name and generate count on the top area of the image generation page.
